# Contributing to LOCKON SDA

First off, thank you for your interest in **LOCKON: System Diagnostic Analyzer**!

## Repository Status: Read-Only / Legacy

This repository serves as a **historical showcase** and **proof-of-concept (Demo)** for the original PowerShell-based architecture. Because this version is considered complete and legacy, **we are not currently accepting Pull Requests, feature requests, or bug reports for this repository.**

## The Future: Enterprise Edition

The LOCKON project has evolved far beyond this initial demo into a fully-fledged **Enterprise-Grade Suite (SaaS)** featuring Cross-Platform scanning, a Desktop GUI, and a Centralized Web Dashboard.

While the full Enterprise version is currently kept in a private repository and is not yet publicly available for viewing, **we have plans to launch an official website showcasing the product very soon!**

If you are interested in the project, stay tuned!

Thank you for your support and understanding.
