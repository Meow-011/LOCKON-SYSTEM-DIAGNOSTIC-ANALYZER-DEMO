## Legacy Repository

This repository contains the original **PowerShell-only Demo** of LOCKON: System Diagnostic Analyzer. It is maintained strictly for historical reference.

**We are no longer providing security updates or patches for this version.**

## Reporting a Vulnerability

If you find a vulnerability in this legacy codebase, please note that it will not be patched here. We appreciate the responsible disclosure, but as this is a frozen proof-of-concept, no further action will be taken on this repository.

For the upcoming **Enterprise Edition (SaaS)**, a separate and active security policy will be enforced. Thank you for your understanding.
